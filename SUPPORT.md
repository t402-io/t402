* **[Feature Request](https://github.com/t402-io/t402/issues/new?template=feature_request.yml)** - Request a feature

### Before Asking

1. **Search existing resources**
   - Check the [documentation](https://docs.t402.io)
   - Search [existing discussions](https://github.com/t402-io/t402/discussions)
   - Search [existing issues](https://github.com/t402-io/t402/issues)

2. **Gather information**
   - SDK and version you're using
   - Blockchain network(s) involved
   - Error messages and stack traces
   - Minimal code to reproduce the issue

3. **Be specific**
   - Describe what you're trying to achieve
   - Explain what you've already tried
   - Include relevant code snippets
